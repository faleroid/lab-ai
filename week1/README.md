# Simulasi Forward Pass Single Perceptron pada Deep learning

## Deskripsi Program
Program mensimulasikan proses forward pass pada sebuah Single Perceptron secara sederhana. 

Program ini menerima input data, lalu mengalikannya dengan bobot yang diinisialisasi secara acak, menambahkan bias, dan melewatkan hasil akhirnya ke dalam fungsi aktivasi Sigmoid untuk menghasilkan output biner 0 atau 1.

## Prasyarat dan Cara Menjalankan
1. Clone atau salin repository ini.
1. Pastikan Python 3.x sudah terpasang.
2. Buka terminal atau *command prompt*.
3. Ketik python main.py.
